✅ todo

⏰ Productivity
Persistent to-do list with proactive reminders and auto-capture. Surfaces top items at session start, auto-captures action items from Slack, Gmail, Calendar, and Linear, and reminds you of untouched high-priority items before wrapping up.

Example usage

  • "add a to-do to review the spec"
  • "show my to-do list"
  • "mark item 3 as done"
  • "snooze 2 until tomorrow"

Features

  • Auto-capture from Slack, Gmail, Calendar, and Linear
  • Priority-based sorting (P1 urgent → P4 low)
  • Snooze support with natural language ("until Monday", "for 4h")
  • Session-start surfacing — top 5 items shown automatically
  • End-of-session reminders for untouched P1/P2 items
  • Auto-archive completed items after 7 days
  • Staleness alerts for items open 5+ days

How it works

  • Stores items in a local JSON file with UUID, priority, source, and status
  • Reads and writes via Python scripts — no external dependencies
  • Integrates with the start-of-day skill as Section 0 (before Slack/Gmail triage)
  • Automatically marks items done when you complete related actions
