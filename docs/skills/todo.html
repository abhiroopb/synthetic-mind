Todo

skill View on GitHub
Persistent to-do list with proactive reminders and auto-capture from Slack, Gmail, Calendar, Linear.

What it does

Todo provides a persistent task tracker stored as a JSON file that the agent manages proactively. It auto-captures action items from other skills (start-of-day triage, emails, Slack messages, Linear issues), surfaces open items at session start, and reminds you of untouched high-priority items before a session ends. Items support priorities (P1–P4), snoozing, due dates, and source references for one-click context.

Usage

The skill runs proactively — surfacing items at session start and auto-capturing new ones. You can also interact with it directly.

Key commands:

  • "add a to-do to review the spec" — manually add an item
  • "show my to-do list" — display all open items
  • "done 1" — mark item #1 as complete
  • "snooze 2 until tomorrow" — snooze an item
  • "prioritize 3 as P1" — change priority
  • "delete 4" — remove an item

Proactive behaviors

  • Session start: Surfaces top 5 open items sorted by priority and due date
  • Auto-capture: Silently creates to-do items from skipped/saved triage items, emails needing replies, Slack questions, assigned issues
  • Auto-complete: When you complete an action matching a to-do, it marks the item done automatically
  • End-of-session: Reminds you of untouched P1/P2 items before wrapping up
  • Staleness alerts: Items open for 5+ days get flagged and bumped up one priority level
  • Auto-archive: Completed items are removed after 7 days

Data schema

Each item tracks: id, title, source (Gmail/Slack/Linear/etc.), source_ref (link or ID), priority (1–4), status (open/in-progress/done/snoozed), due date, snooze_until, timestamps, and context notes.

Why it was created

Action items surface across many tools — Slack, Gmail, Linear, Google Docs, Figma — but none of them talk to each other. Items get lost between sessions. This skill creates a single unified task list that captures items from all sources and proactively reminds you, so nothing falls through the cracks.
