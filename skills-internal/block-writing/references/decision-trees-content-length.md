# Decision Trees: Content Length

## Content Length Decision Tree

```
How much content do you need?

├── ≤15 characters → Buttons (compact), labels, navigation
│   "Save", "Edit", "Settings"
│
├── 15-25 characters → Button labels, input labels, headings
│   "Save changes", "Email address"
│
├── 25-100 characters → Helper text, toast messages, brief descriptions
│   "We'll send updates here", "Payment sent to John"
│
├── 100-200 characters → Modal body, error explanations, instructions
│   Break into digestible chunks; use clear, simple language
│
└── >200 characters → Help articles, legal content, detailed explanations
    Use headers; break into logical sections; consider progressive disclosure
```
