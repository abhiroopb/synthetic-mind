# Google Drive

> Interact with Google Drive, Docs, Sheets, and Slides — search, read, write, share, and manage files.

## What it does

This skill provides comprehensive access to Google Drive, Docs, Sheets, and Slides via a local CLI. It supports searching and listing files, creating and reading documents, writing formatted markdown to Docs, reading and writing spreadsheet data, managing presentations, handling file sharing and permissions, and working with comments and revisions. All commands output JSON.

## Usage

Use when you need to search files, read or write Google Docs, work with spreadsheets, create presentations, manage sharing, or perform any Google Drive file operation.

## Examples

- "Search Google Drive for the quarterly report"
- "Read the contents of this Google Doc and summarize it"
- "Append a new row to the feature tracking spreadsheet"

## Why it was created

Google Workspace is central to document collaboration, but interacting with it programmatically usually requires complex API setup. This skill wraps the full Google Drive, Docs, Sheets, and Slides APIs into simple CLI commands for seamless agent integration.
