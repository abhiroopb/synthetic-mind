# Product

> Search and read product requirements and feature architecture documentation from a central repository.

## What it does

Product provides a CLI tool to search, read, and browse a product documentation repository. You can search for documents by keyword, read specific files, and browse the directory tree to discover what documentation exists. It defaults to searching markdown files and supports JSON output for automation.

## Usage

Invoke when you need to find existing product requirements, look up end-to-end feature architecture, or browse product documentation.

**Trigger phrases:**
- "Search product docs for payments architecture"
- "Read the spec for feature X"
- "Browse the product repo"

## Examples

- `"Search product docs for 'checkout flow'"`
- `"Show me the directory tree of the product repo"`
- `"Read the payments architecture doc"`

## Why it was created

Product requirements and feature specs are spread across a repository that can be hard to navigate. This skill provides fast, targeted access to product documentation without needing to clone or browse the repo manually.
