# Reviewing Calendar

> Review Google Calendar events inline with surrounding context, conflict detection, and RSVP actions.

## What it does

Reviewing Calendar presents calendar events one at a time in chat, showing a 3-hour window before and after each event so you can see surrounding context and detect scheduling conflicts. Each event is displayed in a visual format with status icons, a highlighted box for the target event, and conflict annotations. You can accept, decline, tentatively accept, or skip each event, and it supports batch actions like "accept the next 3."

## Usage

Invoke when you want to review calendar invites, check your schedule, triage pending events, or RSVP to calendar events without leaving your terminal.

**Trigger phrases:**
- "Review my calendar"
- "Check my calendar invites"
- "Show my schedule for this week"
- "Triage pending calendar events"

## Examples

- `"Review my pending calendar invites for this week"`
- `"Show me my schedule for tomorrow"`
- `"Check for conflicts on Wednesday"`

## Why it was created

Reviewing calendar events in a browser requires context-switching and lacks conflict detection at a glance. This skill brings calendar review into the agent workflow with visual context, conflict highlighting, and inline RSVP actions.
