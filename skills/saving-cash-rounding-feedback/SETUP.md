# Saving Cash Rounding Feedback — Setup

## Prerequisites

- **gdrive skill**: Required for reading and writing to the tracking Google Doc. Install via `amp skill add gdrive` or equivalent.
- **slack skill**: Required when feedback source is a Slack message. Install via `amp skill add slack` or equivalent.
- **`sq curl`**: Required for Regulator API lookups to resolve merchant tokens from seller emails.
- **Google Doc access**: The feature feedback tracking doc (see SKILL.md for doc ID) must be editable by your Google account.
