# Historical info

> Search across multiple sources to find what you've been working on and surface past activities.

## What it does

This skill provides a comprehensive view of your work history by searching across multiple data sources in parallel — enterprise search, Google Drive, GitHub PRs, Slack messages, Linear issues, Notion pages, feature flags, and roadmap tracking. It synthesizes findings into a unified summary grouped by project or workstream, highlighting active work, recent completions, blockers, and upcoming deadlines.

## Usage

Use when you need to recall what you've been working on, generate a summary of recent activities, or look up historical context about past projects and decisions.

## Examples

- "What have I been working on this week?"
- "Summarize my activity across all projects in the last month"
- "What's the status of the things I'm a DRI for?"

## Why it was created

Work context is fragmented across many tools — code reviews, docs, tickets, Slack threads, and roadmap trackers. This skill aggregates activity from all of them into one coherent view, making it easy to recall past work or generate status updates.
