# Rewrite and Correction Examples

Use these when converting text into concise, direct, concrete output.

## Rewrite a long note

User ask: "Rewrite this update in jack style."

Input:
"Hi team, just wanted to share that we made some meaningful progress across a number of workstreams this week, but we still have a few concerns and we are thinking through what to do next."

Jack-style response:

`good progress this week.`

`main concern is onboarding drop after step 2.`

`isolate cause today, test fix tomorrow, report by 5pm.`

## Verbose -> corrected

Too verbose:
"Hope you're doing well. I wanted to take a quick moment to say that I really appreciate all the hard work everyone has been doing and I think there are many opportunities ahead of us."

Corrected:

`great work.`

`we have momentum.`

`fix reliability and ship the top customer request next.`

## Hedged/soft -> corrected

Too hedged:
"I might be wrong here, but perhaps we should maybe consider revisiting this decision at some point if people think it could be useful."

Corrected:

`let's revisit this friday with data.`

`if value is unclear, we stop it.`

## Generic -> concrete

Too generic:
"we should align stakeholders and improve communication before moving forward."

Corrected:

`alex schedules a 30-minute decision review by thursday 2pm.`

`sam publishes one-page decision notes by thursday 5pm.`

`if no critical objections by friday noon, we execute.`
