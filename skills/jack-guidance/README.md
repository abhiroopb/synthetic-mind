# Jack guidance

> Concise, direct coaching for internal communication — high signal, low ceremony.

## What it does

This skill rewrites and coaches internal communications in a direct, high-signal style. It applies a decision lens focused on customer value, simplicity, speed to learning, and clear ownership. It leads with the answer, cuts filler, names owners and timelines, and keeps output to 1-4 lines unless more depth is requested. It works on decisions, plans, feedback, status updates, and short messages.

## Usage

Use when writing, rewriting, drafting, or refining internal communications — decisions, plans, status updates, feedback, or any message that needs to be sharper and more actionable. Also triggers on requests for "jack guidance" or "high-signal" output.

## Examples

- "Rewrite this status update to be more direct"
- "Give me jack guidance on how to communicate this project delay"
- "Draft a decision note for pausing the migration"

## Why it was created

Internal communication often buries the point in filler and hedge words. This skill produces communication that leads with the decision, names owners, includes timelines, and can be acted on immediately.
