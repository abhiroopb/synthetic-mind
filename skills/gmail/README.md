# Gmail

> Manage Gmail — search, read, send, draft, and organize emails with labels and filters.

## What it does

This skill provides full Gmail management through an MCP server. It supports searching emails using Gmail search syntax, reading message content, sending and drafting emails, managing labels on messages, and creating or managing Gmail filters. Authentication is handled via OAuth with tokens stored securely in the system keyring.

## Usage

Use when you need to search, read, send, draft, or manage emails. Also use for creating filters, managing labels, or organizing your inbox.

## Examples

- "Search for unread emails from alice@example.com"
- "Draft a reply to the latest email about the Q1 review"
- "Create a filter to auto-label emails from the team mailing list"

## Why it was created

Email is a core communication tool but managing it requires constant context-switching. This skill brings Gmail into the agent workflow, enabling automated triage, drafting, and organization without opening a browser.
