# Cash Rounding Responder

> Triages and drafts replies to shared team inbox emails since the last run.

## What it does

Automatically searches for new inbound emails to a shared team inbox, filters out already-replied threads, classifies each email by category (activation request, troubleshooting, feature request, general inquiry), and drafts context-aware replies using an approved response template. Presents each draft for user approval before creating Gmail drafts — never auto-sends. Tracks the last run timestamp to avoid reprocessing.

## Usage

Use when you need to triage and respond to emails in a shared team inbox. The skill remembers when it was last run and only processes new emails.

Trigger phrases:
- "Respond to the team inbox emails"
- "Triage the shared inbox"
- "Handle the incoming emails"

## Examples

- "Check for new emails since last run and draft responses"
- "Triage the team inbox and show me what needs a reply"
- "Process the shared inbox emails from the last week"

## Why it was created

Shared team inboxes accumulate emails that require similar responses. This skill automates the triage and drafting process while keeping the human in the loop for approval, turning a 30-minute chore into a 5-minute review.
