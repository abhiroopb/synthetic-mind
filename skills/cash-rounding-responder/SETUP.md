# Cash Rounding Responder — Setup

## Prerequisites

- **gdrive skill**: Required for reading the penny elimination doc template. Install via `amp skill add gdrive` or equivalent.
- **Gmail MCP**: Required for searching, reading, and drafting email replies. Ensure Gmail MCP tools are configured.
- **Feature reference doc access**: The Google Doc (see SKILL.md for doc ID) must be accessible to your Google account.
