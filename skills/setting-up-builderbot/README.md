# Setting Up Builderbot

> Guide users through installing and configuring the CLI for an AI task orchestration system.

## What it does

This skill walks you through setting up a task orchestration CLI tool that routes work to AI agent executors based on task labels, status, and attached artifacts. It covers prerequisites (access permissions), building the CLI from source, adding it to your PATH, and common pitfalls to avoid. Once set up, you can use the companion `creating-builderbot-tasks` skill for actual task operations.

## Usage

Invoke when you need to install or configure the task orchestration CLI for the first time, or when troubleshooting setup issues.

**Trigger phrases:**
- "Set up builderbot"
- "Install the builderbot CLI"
- "Configure builderbot"
- "I need help getting builderbot working"

## Examples

- `"Help me set up builderbot from scratch"`
- `"I installed builderbot but the CLI isn't found"`
- `"Walk me through builderbot setup"`

## Why it was created

Getting a CLI tool built from source and properly configured involves multiple steps that are easy to get wrong. This skill provides a guided, minimal setup path with common pitfall warnings so you can get productive quickly.
