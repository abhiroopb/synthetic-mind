# Controlling Computer

> Control your Mac via AppleScript and shell commands — apps, windows, volume, brightness, and system actions.

## What it does

Provides macOS automation through AppleScript and shell commands. Capabilities include opening/quitting/switching apps, managing window positions and sizes, adjusting volume and brightness, toggling dark mode, reading clipboard contents, taking screenshots, querying system info (battery, disk, CPU, Wi-Fi), and simulating keyboard/mouse input. Always confirms before destructive actions like restart or shutdown.

## Usage

Use when you need to automate macOS actions, manage windows, change system settings, or query system information.

Trigger phrases:
- "Open Slack"
- "Set volume to 50%"
- "Toggle dark mode"
- "Tile these two apps side by side"
- "What's my battery level?"

## Examples

- "Mute the volume and turn on dark mode"
- "Tile Safari and VS Code side by side"
- "Take a screenshot of the current screen and save to Desktop"

## Why it was created

Switching between apps, adjusting settings, and managing windows interrupts flow. This skill lets you control your Mac without leaving the terminal, keeping hands on keyboard for common system actions.
