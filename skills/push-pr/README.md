# Push PR

> Push the current branch and create a draft PR with an AI-generated description.

## What it does

Push PR is a streamlined, opinionated workflow that pushes your current Git branch to origin and creates a draft pull request in one step. It analyzes the diff and commit history to generate a PR description, follows any existing PR template in the repo, and uses conventional commit style for the PR title. All PRs are created as drafts.

## Usage

Invoke when you want to quickly push your branch and open a draft PR without manually writing the description.

**Trigger phrases:**
- "Push PR"
- "Create PR"
- "Push and create PR"

## Examples

- `"Push PR"`
- `"Push this branch and create a draft PR"`
- `"Create a PR for my current changes"`

## Why it was created

Pushing a branch and creating a PR with a good description involves multiple manual steps. This skill collapses them into a single command with auto-generated descriptions, so you can get code up for review faster.
