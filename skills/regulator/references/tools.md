# Available Tools

| Tool | Purpose |
|------|---------|
| `mcp__mcp-regulator__find_merchant_by_business_name` | Two-step search: find merchant by business name |
| `mcp__mcp-regulator__query_merchant` | Get merchant details by token |
| `mcp__mcp-regulator__query_location` | Query locations with filters (name, email, coordinates, MCC, status) |
| `mcp__mcp-regulator__query_location_hardware` | Hardware devices at a location |
| `mcp__mcp-regulator__query_location_capital` | Capital plan details for a location |
| `mcp__mcp-regulator__query_merchant_employees` | Paginated employee list for a merchant |
| `mcp__mcp-regulator__query_customer` | Customer data by type (STANDARD, MOBILE_APP, CAPITAL, GIFT_CARD, REMITTANCE) |
| `mcp__mcp-regulator__query_payments` | Payments with filters (date, status, entry method, risk, dispute, etc.) |
| `mcp__mcp-regulator__query_payment_details` | Detailed payment info for up to 20 payment tokens |
