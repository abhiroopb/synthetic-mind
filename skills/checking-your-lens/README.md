# Checking Your Lens

> Add a final judgment pass so the output sounds like you, reflects your priorities, and fits the real context.

This skill is for the last 10% of quality: the part where a draft stops sounding generic and starts sounding like a real operator wrote it.

## What it does

- reviews a draft against the actual repo, workstream, or project context
- cuts generic advice and over-explanation
- checks that the answer leads with a recommendation
- makes the final output feel grounded, direct, and usable

## When to use it

- "give this one last pass"
- "make this sound more like me"
- "tighten this before I send it"
- "check this against the real context"

## Why it matters

Most weak AI output is not wrong. It is just too generic. A good judgment pass fixes that by checking voice, context, trade-offs, and actionability right before the output goes out.
