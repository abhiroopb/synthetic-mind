# Dev guides

> Search and retrieve internal developer documentation as readable markdown.

## What it does

This skill searches and fetches developer documentation from an internal docs platform. It supports full-text search with optional index filtering, and can fetch any documentation page as markdown for easy reading and context gathering. It also provides commands to sync indexes, list documents, and view statistics.

## Usage

Use when you need to find internal developer documentation, look up guides, or fetch a specific docs page. Invoke with search queries or direct URLs.

## Examples

- "Search dev guides for acceptance test setup"
- "Find documentation about the web monorepo"
- "Fetch the guide on end-to-end testing"

## Why it was created

Developer documentation is scattered across many pages and hard to search manually. This skill provides fast, CLI-based access to the full docs corpus so you can find answers without leaving your workflow.
