# Airtable

> Integrate with Airtable's cloud database platform for managing bases, tables, and records.

## What it does

Provides comprehensive Airtable API integration including full CRUD operations on records, advanced filtering with formula syntax, batch processing (up to 10 records per request), pagination, sorting, and webhook subscriptions for real-time change notifications. Supports 20+ field types including attachments, linked records, and formulas.

## Usage

Use when you need to create, read, update, or delete Airtable records, query bases, filter data, or manage table schemas. Requires a Personal Access Token for authentication.

Trigger phrases:
- "Search Airtable for..."
- "Create a record in Airtable"
- "Update the status in our Airtable base"
- "Filter Airtable records where..."

## Examples

- "List all active records in the project tracker base"
- "Create a new record with name 'Q1 Review' and status 'Pending'"
- "Filter the feature requests table for high-priority items"

## Why it was created

Airtable is used for tracking roadmaps, feature requests, and project data. This skill eliminates the need to switch to the Airtable UI for quick lookups and updates, making data management accessible from the command line.
