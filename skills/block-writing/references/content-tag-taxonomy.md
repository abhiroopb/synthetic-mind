# Content Tag Taxonomy

## Tag format
- Hyphens: `brand-b` not `brand b`; lowercase; descriptive: `onboarding-flow` not `onboard`
- Apply 3-7 tags per document; always include at least one product tag and one content type tag

## Priority order
1. Product (which your company product?)
2. Content type (what kind of writing?)
3. Writing element (what aspect of writing?)
4. Audience/context (who and when?)
5. Channel/format (where does it appear?)
6. Topic/domain (what subject matter?)

## Product tags
**your company**: `brand-b` `brand-a` `brand-c` `brand-d` `brand-e` `brand-f` `company`
**Brand B**: `brand-b-card` `brand-b-pay` `brand-b-taxes` `brand-b-borrow` `brand-b-investing` `brand-b-savings` `brand-b-afterpay`
**Brand A**: `square-pos` `square-online` `square-terminal` `square-register` `square-payments` `square-payroll` `square-marketing` `square-loyalty` `square-appointments` `square-restaurants` `square-retail`

## Content type tags
**UI**: `ui-copy` `error-messages` `success-messages` `onboarding-flow` `empty-states` `loading-states` `form-labels` `button-text` `tooltips` `modal-content` `navigation-labels`
**Marketing**: `marketing-copy` `growth-messaging` `promotional-copy` `email-marketing` `social-media` `advertising-copy` `landing-pages`
**Transactional**: `transactional-comms` `email-notifications` `sms-notifications` `push-notifications` `in-app-messages` `receipts`
**Support/Docs**: `help-content` `faq-content` `troubleshooting` `how-to-guides` `api-documentation` `legal-copy` `policy-documents`

## Writing element tags
**Voice**: `voice-and-tone` `brand-personality` `conversational-tone` `empathetic-tone` `confident-tone`
**Style**: `style-guide` `grammar-rules` `capitalization` `punctuation` `numbers-formatting` `contractions`
**Standards**: `accessibility` `plain-language` `localization` `seo-guidelines` `character-limits`
**Terms**: `terminology` `brand-guidelines` `technical-terms`

## Audience tags
**External**: `customer-facing` `merchant-facing` `developer-facing` `media-facing`
**Internal**: `internal-comms` `training-materials` `process-documentation`
**Context**: `first-time-users` `power-users` `mobile-experience` `emergency-situations`

## Channel tags
**Digital**: `email` `sms` `push-notifications` `in-app` `website` `social-media` `chat-support`
**Support**: `help-center` `knowledge-base` `phone-support`
**Format**: `long-form` `short-form` `microcopy` `interactive` `visual-content`

## Topic tags
**Finance**: `payments` `banking` `lending` `investing` `savings` `taxes` `fraud-prevention` `cryptocurrency`
**Tech**: `mobile-technology` `web-technology` `data-privacy` `cybersecurity` `user-experience`
**Business**: `small-business` `e-commerce` `retail` `restaurants` `inventory-management`
