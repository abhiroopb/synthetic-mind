# Brand B Terminology

Always use exact capitalization and spelling. Check "Avoid" column for deprecated terms.

## Product names

| Use | Avoid | Notes |
|-----|-------|-------|
| **Brand B** | "the Brand B", "BrandB", "brand b" | No article; always capitalize both words |
| **Brand B Card** | "Brand B Card" (alone), "brand b card" | Full name on first reference; "card" OK after |
| **Brand B Pay** | "Cash Pay", "BrandB Pay" | — |
| **Brand B Taxes** | "Cash Taxes", "BrandB Taxes" | — |
| **Brand B Investing** | "Cash Investing", "BrandB Stocks" | — |

## Account and identity

| Use | Avoid | Notes |
|-----|-------|-------|
| **$cashtag** | "cash tag", "CashTag", "Cashtag", "cash-tag" | Lowercase; include $ in actual user tags |
| **Brand B balance** / balance | "wallet", "account balance" | "Cash balance" only when "Cash" is the tab name in UI |

## Money movement

| Use | Avoid | Notes |
|-----|-------|-------|
| **add money** (to Brand B balance) | "deposit", "load funds", "cash in" | Bringing money into Brand B |
| **send** (to someone) | "transfer", "pay", "remit" | Primary P2P action verb |
| **withdraw** (to bank) | "cash out" ⚠️ deprecated, "transfer out" | "Cash out" is old terminology — never use |
| **request** (payment from someone) | "ask for money", "charge" | — |

## Feature names

| Use | Avoid | Notes |
|-----|-------|-------|
| **direct deposit** (lowercase) | "Direct Deposit", "DD" | Lowercase unless starting a sentence |
| **Cash Boost** | "cash boost", "Boost", "CashBoost" | Capitalize the specific feature |
| **Bitcoin** | "BTC", "bitcoin" (user-facing) | Full name in customer content; BTC OK in charts/graphs |
