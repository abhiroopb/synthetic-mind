# Brand C on Brand B: Brand Strategy and Content Guidelines

## Brand Position
- Brand C is Brand B's umbrella brand for lending products (excluding Borrow)
- Represents all pay-over-time experiences within Brand B
- Don't use: "Brand B Brand C" in product

**Brand voice**: Clear and direct; benefits-focused; user-friendly; confidence-building
**Messaging approach**: Lead with benefits; progressive disclosure; clear next steps; consistent terminology

## Naming and Communication Principles

**User-first messaging**: Lead with clear, immediate benefits; use progressive disclosure; keep language simple; focus on user outcomes

**Trust building**: Be transparent about how Brand C works; use clear, straightforward language; build confidence through reliability

**Information design**: Layer complexity appropriately; present information at point of need; keep core messages evergreen

## Content Guidelines

**Be precise and accurate**:
- Verify all claims before making them
- Use correct terminology consistently
- Balance messaging when showing multiple actions

**Consider user context**:
- Write for new users who don't know Brand C
- Place eligibility messaging strategically
- Maintain clarity across all user types

**What works**:
- Explains benefit clearly; uses "with Brand C" appropriately; balances multiple use cases; avoids redundancy; clear for new users

**Avoid**:
- Making assumptions about number of brands/stores
- Using specific payment numbers/terms in UI that should be evergreen or too early in the flow
- Creating redundancy between headers and subheads
