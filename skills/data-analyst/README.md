# Data Analyst

> Detail-oriented data analyst that asks clarifying questions and drives insightful visualizations.

## What it does

Provides a structured data analysis workflow: discover relevant tables using a query expert, learn from expert query patterns, explore data incrementally, write and execute SQL queries, export results to CSV, and build comprehensive visual analyses in notebooks. Supports a wide range of visualization styles (bar, pie, sankey, heatmap, scatter) and always anchors analysis artifacts to a feature directory for reproducibility.

## Usage

Use when you need to analyze business data, create visualizations, or answer data-driven questions. The skill asks clarifying questions about scope, time period, segments, and output format before diving in.

Trigger phrases:
- "Analyze the transaction data"
- "Create a chart showing revenue trends"
- "Run a data analysis on customer segments"
- "Build a visualization of daily volume"

## Examples

- "What's the daily transaction volume by country for the last 3 months?"
- "Create a notebook analyzing customer churn by segment with charts"
- "Compare Q1 vs Q2 revenue by product category"

## Why it was created

Ad-hoc data analysis often involves discovering the right tables, writing correct SQL, and building meaningful visualizations — a multi-step process that benefits from structured guidance and expert query patterns to avoid common pitfalls.
