# Glean

> Search enterprise knowledge, documents, and people directory via Glean.

## What it does

This skill provides access to an enterprise knowledge search platform that indexes content across Confluence, Google Drive, Slack, Jira, GitHub, and more. It supports full-text search with datasource filters, conversational Q&A powered by AI, reading full document content from search results, and discovering specialized AI agents for domain-specific questions.

## Usage

Use when you need to search company documentation, find people, ask questions grounded in the internal knowledge base, or read documents discovered through search.

## Examples

- "Search for documentation about the deployment pipeline"
- "Ask Glean: what's the process for requesting a new service?"
- "Find all Confluence pages about cash rounding"

## Why it was created

Internal knowledge is scattered across many tools — wikis, docs, Slack, issue trackers. This skill provides a unified search layer that finds answers across all of them, eliminating the need to manually search each tool individually.
