# Block Data

> Query business metrics, dashboards, and data permissions via the Metric Store.

## What it does

Provides access to trusted business metrics (KPIs, revenue, GPV, active users, transaction volumes) through a centralized Metric Store. Supports searching for metrics by keyword, fetching metric data with filters and date ranges, discovering and executing Looker/Mode/Tableau dashboards, and checking table-level data access permissions. Enforces a structured workflow: search → choose → validate dimensions → fetch.

## Usage

Use when asked about business KPIs, revenue numbers, GPV, active users, or other company metrics. Also use for discovering dashboards or checking data access permissions.

Trigger phrases:
- "What's our GPV?"
- "Show me active users last quarter"
- "Find dashboards about revenue"
- "Check my permissions for this table"

## Examples

- "What metrics are available for mobile app revenue?"
- "Show me daily transaction volume for the last 30 days"
- "Find Looker dashboards related to payment processing"

## Why it was created

Business metrics are scattered across multiple dashboards and data sources. This skill provides a single entry point to search, discover, and query trusted metrics without context-switching between analytics tools.
